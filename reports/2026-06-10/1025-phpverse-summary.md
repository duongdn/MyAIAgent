### 🎯 Talks nổi bật

**"My AI Writes Perfect* PHP" — Ashley Hindle (Fuel / Laravel Boost)**
Talk được chú ý nhất. Demo AI setup viết PHP đúng style, conventions, tự viết test và cập nhật docs — code mà dev "không phải chiến đấu với nó". Liên quan trực tiếp workflow Laravel hàng ngày.

**"How AI Is Changing the Programmer World" — Jeffrey Way (Laracasts)**
Nhìn rộng về vai trò developer trong thời AI — không chỉ về tool mà về tư duy và tương lai nghề. Jeffrey Way là người dạy Laravel/PHP cho cả cộng đồng nên talk này được đặc biệt chú ý.

**"WordPress Is Dead, Long Live WordPress!" — Jonathan Bossenger (Automattic)**
Bác bỏ 10 myth phổ biến:
- PHP 8.0-8.3 chính thức, beta 8.4-8.5
- Block themes thay PHP templates thuần
- WP 6.9: nhanh hơn 2.8-5.8%, lên đến +23% trên PHP 8.5
- **WP Packages** (3/2026): Composer-based WordPress workflow
- Interactivity API cho reactive state, Block Bindings API (WP 6.5) native custom fields
- REST API xử lý exception đúng cách

**"Packagist Internals" — Nils Adermann (Packagist co-founder)**
Behind-the-scenes hạ tầng Composer/Packagist — đặc biệt nhạy cảm sau vụ supply chain near-miss CVE-2026-45793 gây xôn xao tuần trước.

**"Running a Large Open-Source Project Like Symfony" — Fabien Potencier**
Vận hành Symfony quy mô lớn, duy trì community và kết nối ecosystem.

**PHP Foundation Keynote — Elizabeth Barron**
3 điểm: Foundation làm gì, initiatives 2026, cách community đóng góp.
