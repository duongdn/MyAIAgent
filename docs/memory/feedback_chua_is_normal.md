"Chưa" (meaning "not yet" in Vietnamese) in the task log confirmation column (col P) is the normal default state across most spreadsheets. Do NOT flag it as an alert.

**Why:** User pointed out that most task logs have the same "Chưa" status — singling out one project for it is incorrect.
**How to apply:** When reviewing Google Sheets task logs, ignore column P "Chưa" as a status indicator. Only flag actual hour discrepancies or missing data as alerts.
