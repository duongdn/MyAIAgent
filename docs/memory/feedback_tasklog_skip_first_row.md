type: feedback

Never write task log entries on the first empty row of a day. Always skip to at least the second empty row.

**Why:** First row is reserved (user had to manually move the entry down). The first row after a date header may be used for other purposes.

**How to apply:** When finding empty rows after today's date row in the Paturevision Google Sheet, skip the first empty "Task dự án" row and write to the second one instead.
