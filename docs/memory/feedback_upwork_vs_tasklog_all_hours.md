Upwork tracks total actual time worked. When comparing Upwork hours vs task log, sum ALL rows for the employee — both "Task dự án" (official) and "Part-time" rows in column A.

**Why:** Task log official-only showed 36h for LeNH/Rory, but Upwork showed 40:10h. Adding the 4.17h part-time row matched exactly (40.17h).
**How to apply:** In weekly report Upwork comparison, use total hours (official + PT) from task log to match against Upwork tracked hours. Only filter to "Task dự án" for the daily report official hours check.
